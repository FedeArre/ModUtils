# Frequently Asked Questions

## Can it load cars?
It was never tried, but is techically possible to load all the custom parts of a car using it.

## My part painting does not work!
If there are no errors on the log about the issue, your issue is probably related to UV issues. Make sure if everything is on UV channel 0, the submeshes are correct respect to the type choosen and your part does not work after attaching (Some weird cases can make a part not work until attached!). If you can't fix come to the official My Garage discord server and ask for help, we might be able to troubleshoot the issue.

## What is developer log?
SPL.DEVELOPER_LOG is an internal variable used for developing SimplePartLoader that provides very useful information about dummy parts cloning and other features. If you are having trouble with things not working as expected this might help you. Remember to set it to false before releasing your mod!

## Does it work on survival?
SimplePartLoader works on survival, is up to developers to choose if their parts can appear on survival or no.
